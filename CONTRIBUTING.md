# Contributing

Thanks for helping improve Reconeer!

## Quick rules
- Be respectful (see CODE_OF_CONDUCT.md).
- Keep changes focused and well-tested.
- Avoid adding heavy dependencies unless clearly justified.

## Development
Requirements:
- Go 1.22+

Run locally:
```bash
go test ./...
go run ./cmd/reconeer -d example.com
```

## PR checklist
- [ ] Code compiles (`go test ./...`)
- [ ] README/docs updated if behavior changes
- [ ] Added/updated tests when applicable
- [ ] Clear description + screenshots/logs for UX changes

## Good first issues
Look for issues labeled **good first issue** / **help wanted**.
